# Branch Workflow / Як працювати з гілками:

1. Clone the repository/Склонуйте репозиторій (вперше):
    git clone https://github.com/arstrongDV/CoffeeJoy.git
    cd CoffeeJoy

# Create your own branch / Створіть нову гілку від main(на якій буде зберігатися лише ваша частина коду)

1. git checkout -b your_branch_name (називайте гілки назвами ваших секцій)
    Example: git checkout -b header-section

2. Check your current branch(перевірте чи ви знаходитесь на вашій гілці): 
    git branch

3. Add your changes(Додайте свої зміни після редагування коду): 
    git add .

4. Commit your work(Зробіть коміт з коротким, зрозумілим описом):
    git commit -m "Add hero section layout and styles"

5. !!!! Push your branch to GitHub !!!! (Відправте свою гілку на віддалений репозиторій):
    git push -u origin your_branch_name
    Example: git push -u origin header-section

6. Push your changes(Надішліть зміни на репозиторій): 
    git push
